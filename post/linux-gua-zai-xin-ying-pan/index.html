Linux挂载新硬盘

· 2019-06-26 · # Linux # 磁盘挂载
  1. 查看新磁盘：sudo fdisk -l，假设要挂载的磁盘为/dev/sdc
  2. 创建分区：fdisk /dev/sdc
  3. 写入文件系统：mkfs.ext4 /dev/sdc
  4. 新建挂载目录：mkdir /mnt/data
  5. 挂载：mount /dev/sdc /mnt/data
  6. 设置开机自动挂载：
vi /etc/fstab
# 在文件末尾添加
/dev/sdc /mnt/data ext4 defaults 0 0
