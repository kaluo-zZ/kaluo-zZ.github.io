Windows安装cocoapi时出现错误

2019-06-11

cocoapi for windows: -> or here.
错误提示：

PS D:\Files\COCO\cocoapi\PythonAPI> make
python setup.py build_ext --inplace
running build_ext
building 'pycocotools._mask' extension
error: Microsoft Visual C++ 14.0 is required. Get it with "Microsoft Visual C++ Build Tools": https://visualstudio.microsoft.com/downloads/
make: *** [Makefile:3: all] Error 1

解决方法：
下载安装Visual C++ Build Tools 2015 ->

windows安装coco python api (python3.6环境)：

  1. cd coco/PythonAPI
  2. 删除setup.py中的“Wno-cpp”和"Wno-unused-function";
  3. python setup.py build_ext --inplace
  4. python setup.py build_ext install
