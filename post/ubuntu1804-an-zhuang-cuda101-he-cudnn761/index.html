Ubuntu18.04安装cuda10.1和cudnn7.6.1

2020-06-01

安装前准备：cuda_10.1.168_418.67_linux.run和cudnn-10.1-linux-x64-v7.6.1.34.tgz

  1. sudo sh cuda_10.1.168_418.67_linux.run
  2. vi ~/.bashrc: 写入
    export PATH=$PATH:/usr/local/cuda-10.1/bin
export LD_LIBRARY_PATH=$LD_LIBRARY_PATH:/usr/local/cuda-10.1/lib64
export CUDA_HOME='/usr/local/cuda-10.1'
  3. source ~/.bashrc
  4. 测试安装是否成功：nvcc -V:
     nvcc: NVIDIA (R) Cuda compiler driver
 Copyright (c) 2005-2019 NVIDIA Corporation
 Built on Wed_Apr_24_19:10:27_PDT_2019
 Cuda compilation tools, release 10.1, V10.1.168
  5. 解压：tar -xvf cudnn-10.1-linux-x64-v7.6.1.34.tgz
  6. 执行命令：
    sudo cp cuda/include/cudnn.h /usr/local/cuda/include/
sudo cp cuda/lib64/libcudnn* /usr/local/cuda/lib64/
sudo chmod a+r /usr/local/cuda/include/cudnn.h
sudo chmod a+r /usr/local/cuda/lib64/libcudnn*
  7. 测试是否安装成功：cat /usr/local/cuda/include/cudnn.h | grep CUDNN_MAJOR -A 2
    #define CUDNN_MAJOR 7
#define CUDNN_MINOR 6
#define CUDNN_PATCHLEVEL 1
--
#define CUDNN_VERSION (CUDNN_MAJOR * 1000 + CUDNN_MINOR * 100 + CUDNN_PATCHLEVEL)

#include "driver_types.h"
