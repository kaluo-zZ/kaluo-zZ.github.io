Python属性访问

2019-07-03
  1. __getattr__(self, name)：访问不存在的属性时自动调用
  2. __getattribute__(self, name)：访问存在的属性时自动调用（先调用该方法，查看是否存在该属性，若不存在，调用__getattr__。若重载了__getattribute__方法，则不会自动调用__getattr__方法，需在最后手动添加一行object.__getattribute__(self, name)。）。
  3. __setattr__(self, name)：设置实例对象的一个新的属性时自动调用
  4. __delattr__(self, name)：删除一个实例对象的属性时自动调用

（python中无论实例还是类，都是用__dict__来存储属性和方法）
