Docker相关命令

2019-06-06 docker

  1. 查询容器：docker ps -a
  2. 终止容器：docker stop $(id)
  3. 删除容器：docker rm $(id)
  4. 查询镜像：docker images
  5. 删除镜像：docker rmi $(id)
